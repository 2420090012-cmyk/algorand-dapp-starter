# Algorand Project 🚀

Algorand dApp Starter

## 📖 Description
This repository contains my Algorand project developed as part of my coursework.  
The goal is to build decentralized applications (dApps) using the Algorand blockchain.

- TestNet deployment

- Python / JavaScript
- Pera Wallet
- Algorand TestNet

2. Connect to Algorand TestNet
3. Run the project
